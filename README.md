# Surge Rules

本仓库的规则仅适用于 [Surge](https://nssurge.com/)。

## Rules

| 规则名称 | Raw 链接 |
| --- | --- |
| AI | [AI.list](https://raw.githubusercontent.com/missuo/surge_rules/main/rule/AI.list) |
| BankUS | [BankUS.list](https://raw.githubusercontent.com/missuo/surge_rules/main/rule/BankUS.list) |
| Bybit | [Bybit.list](https://raw.githubusercontent.com/missuo/surge_rules/main/rule/Bybit.list) |
| Cursor | [Cursor.list](https://raw.githubusercontent.com/missuo/surge_rules/main/rule/Cursor.list) |
| WeChat | [WeChat.list](https://raw.githubusercontent.com/missuo/surge_rules/main/rule/WeChat.list) |

## Usage

在 Surge 配置文件中添加规则：

```ini
[Rule]
RULE-SET,https://raw.githubusercontent.com/missuo/surge_rules/main/rule/AI.list,PROXY
```
